# API Portal
### Department of Computer Engineering - University of Peradeniya

This is an open-source API portal of the *Department of Computer Engineering, University of Peradeniya*.

If you are interested in developing apps/websites related to the department's activities, youâ€™re in the right place. You can find the information easily from the APIs here, and focus on making awesome services, apps, websites, etc...

If you are interested in contributing to this, you can find the GitHub repository from [here](https://github.com/cepdnaclk/api.ce.pdn.ac.lk).
